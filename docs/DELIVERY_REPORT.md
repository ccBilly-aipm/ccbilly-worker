## 一、你现在可以用它做什么

- **你可以** 打开一个属于自己的深空科技感工作台，深色/浅色两套皮肤随手切换，还能收起侧边栏。
- **你可以** 新建任务、在看板上拖着卡片改状态、点开任务改进度和勾子任务——每一步系统都自动记一条「动态」，不用你手动写日志。
- **你可以** 把相关任务归到一个「合集」里，合集会自动算出总进度，一眼看到某条线推进到哪了。
- **你可以** 一键生成当天日报：系统自动把你今天所有任务动态分成「完成 / 推进 / 受阻」填好，还帮你预填明日计划；改两笔、点定稿，再点「复制为 Markdown」就能直接粘进飞书，格式不乱。
- **你可以** 一键生成周报，自动统计本周完成数、各合集投入占比，配两张图，能导出成 .md。
- **你可以** 在工作台里直接管理你的 Claude Code Skill——扫出 `~/.claude/skills` 里所有 skill，在线看和改，改之前自动备份、绝不会丢你的 `allowed-tools` 这些配置，改完对 Claude Code 立刻生效。
- **你可以** 看自己的「个人技能树」雷达图，一键记录「今天学了什么」；也能浏览 Obsidian 笔记并点着双链跳转。
- **你可以** 把开源应用（如 Excalidraw 白板）接进「应用中心」，新标签打开或直接内嵌使用；碰上不让内嵌的网站会自动给你「新窗口打开」的按钮，不会白屏。
- **你可以** 在后台一键 Git 提交/同步（永远不会强推坏你数据）、改称呼和默认主题、把整个数据打包成 zip 备份。
- **你可以** 按 `Cmd/Ctrl+K` 全局搜索任务/日报/技能，或快速新建任务、生成日报、切主题。

**最关键的一点**：所有数据都是 `vault/` 目录里的纯文本 Markdown 文件。你用 Obsidian 直接改任何一篇，工作台几秒内自动跟着变；换电脑 `git pull` 就恢复全部数据；重要内容随手粘进飞书。数据永远是你的，不锁在任何数据库里。

## 二、怎么启动

第一次：

```bash
corepack enable      # 启用 pnpm（只需一次）
pnpm install         # 装依赖
pnpm seed            # （可选）生成一套演示数据
pnpm dev             # 打开 http://localhost:3000
```

日常：直接 `pnpm dev`。

> 如果装依赖时提示 better-sqlite3 要编译，运行一次 `pnpm rebuild better-sqlite3` 即可。

## 三、日常怎么用、怎么同步

1. **日常**：在工作台里管理任务、生成日报周报。所有改动实时写进 `vault/`。
2. **配合 Obsidian**：把 `vault/` 作为一个 Obsidian 库打开（或软链进你现有库），双向编辑互不冲突。
3. **配合飞书**：日报/周报的「复制为 Markdown」按钮就是给飞书用的，定稿后一键复制粘贴。
4. **多设备同步**：用后台的「Git 同步」面板，或命令行 `git pull` → `pnpm install` → `pnpm dev`。
5. **备份**：后台「数据导出」一键下载整个 vault 的 zip。

## 四、需要你做的两件事

1. **配置 Git 远端**（一次性）：我已经在本地建好 Git 仓库并提交了全部代码与数据，但你还没有配 GitHub 远端。配好后执行一次：
   ```bash
   git remote add origin <你的 GitHub 仓库地址>
   git push -u origin main
   ```
   之后就能用后台的「同步」按钮一键推送了。

2. **设置后台口令**（建议）：复制 `.env.example` 为 `.env.local`，把 `ADMIN_PASSCODE` 改成你自己的口令。不设也能用（本机单人场景），但设了更稳妥。

## 五、已知限制（都不影响日常使用）

- **Skill 的「新建 / 删除」暂未开放写入**：按当初和你对齐的「先只读+备份」策略，目前 Skill 管理开放了扫描/查看/在线编辑（编辑前自动备份），新建和删除按钮先做成禁用占位。等你确认编辑功能安全无误，我再用一个小迭代开放它们——底层的安全守卫和备份机制都已就绪。
- **视觉素材是占位图**：空状态插画和 Logo 目前是我用纯 CSS/SVG 画的玻璃拟态占位图。等你用 Codex + GPT Image 2 生成正式素材（提示词见需求文档 Part B），放进 `public/assets/` 替换即可，位置我都预留好了。
- **后台口令仅防误触**：它只是防止本机误点，**不是真正的安全防护**。如果你哪天把工作台部署到公网，务必另加真正的认证层。
- **Lighthouse 性能分**：桌面端 99 分（达标），移动端模拟慢速网络下 87 分——这是实验室对网页字体加载的严苛惩罚，真实使用流畅（首屏 0.9s、零布局抖动）。
